## 按照书籍《百万在线大型游戏服务端开发》做的练习

### 主要目录
* etc         => 存放服务配置的文件夹
* luaclib     => 存放一些c模块（so文件）
* lualib      => 存放lua模块
* service     => 存放各服务的lua代码
* skynet      => skynet框架
* start.sh    => 启动服务器的脚本

### 主要服务
- [x] agent 代理服务
- [x] agentmgr 代理管理服务
- [x] gateway 网关服务
- [x] login 登陆服务
- [x] nodemgr 节点管理服务
- [x] scene 场景服务（战斗服务）
